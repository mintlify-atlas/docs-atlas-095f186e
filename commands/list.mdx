`aube list` prints the dependency tree for the current project, showing each installed package and its version. By default it shows only direct dependencies (depth 0), but you can expand the tree to any depth or walk the full transitive graph. Use `ls` as a shorter alias.

## Usage

```bash
aube list [FLAGS] [PATTERN]
```

Pass an optional `PATTERN` to filter output to packages whose name starts with that string (e.g. `aube list '@babel'`).

## Flags

| Flag | Description |
|------|-------------|
| `--depth <DEPTH>` | How deep to render the transitive tree. `0` (default) shows only direct deps. Pass `9999` or `--depth=Infinity` for the full graph. |
| `--json` | Shortcut for `--format json`. Emit a JSON array of package entries. |
| `--parseable` | Shortcut for `--format parseable`. Emit one tab-separated line per package. |
| `--format <FORMAT>` | Output format: `default`, `json`, or `parseable`. Default: `default`. |
| `--long` | Add the store path for each entry (useful for debugging symlink locations). |
| `-P --prod` | Show only production dependencies (skip devDependencies). |
| `-D --dev` | Show only devDependencies. |
| `-g --global` | List globally-installed packages instead of the project's dependency tree. |
